# Sorting-Algorithm-Visualiser
A simple class-based sorting algorithm visualiser built with python and an Extension of https://dev.to/kgprajwal/build-a-sorting-visualizer-in-python-2oej
#
# Algorithms:
- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort
- Heap Sort
- Radix Sort
- Counting Sort
- Bucket Sort
- Shell Sort
